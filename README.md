# dotfiles

## .config

```bash
cd ~/.config
git init
git remote add origin git@github.com:mcquenji/dotfiles.git
git fetch
git pull
```

## .zshrc

```bash
cd ~/.config
ln -s .zshrc ~/.zshrc -f
```
